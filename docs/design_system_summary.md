# Café con Leche - Design System Quick Reference

## Colors
| Name             | Hex     | Use                              |
|------------------|---------|----------------------------------|
| Cream            | #FAF6F1 | Primary background               |
| Warm White       | #FFFDF9 | Card backgrounds                 |
| Linen            | #F0EBE3 | Secondary bg, borders, sidebar   |
| Espresso         | #2C2520 | Primary text, headings           |
| Dark Chocolate   | #3D3530 | Secondary text, labels           |
| Café con Leche   | #C4A882 | Accent, borders, icons           |
| Dulce de Leche   | #D4A574 | Highlight, active states         |
| Soft Terracotta  | #C17F59 | Negative/down/warning            |
| Sage             | #8B9E82 | Positive/up/ok                   |
| Muted Rose       | #B8877A | Secondary accent, critical       |
| Faded Ink        | #6B5E55 | Tertiary text, timestamps        |

## Fonts
- Headings: DM Serif Display (serif)
- Body: Inter (sans-serif)
- Numbers: JetBrains Mono (monospace)

## Cards
- Background: Warm White | Border: 1px Linen | Radius: 12px
- Shadow: 0 1px 3px rgba(44,37,32,0.06)
- Padding: 24px | Gap: 16px

## Number Format (Argentine)
- Currency: $ 1.280,00 (dot thousands, comma decimals)
- Percentage: 17,4%
- Change: +2,4% (Sage) | -0,8% (Terracotta) | - (Café)
